# Contributor Ideas

## Good first issues

- Add keyboard shortcut hints to the command rail.
- Improve empty states in the inspector panels.
- Add a new furniture station type.
- Add a new office preset.
- Add a visible backend reconnect button.
- Add a screenshot gallery to the README.
- Add pytest coverage for `/health` and `/furniture`.

## Advanced contributions

- Persistent database layer.
- Server-side SimAI proxy.
- Streaming output channel.
- Connector permission registry.
- Work product library.
- Global command palette.
- Multi-user collaboration.
- Avatar and animation improvements.
