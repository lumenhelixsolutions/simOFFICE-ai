# Demo Script

## 30-second pitch

"This is SimOffice: a 3D command center for AI agent teams. Instead of managing agents as invisible prompts in a sidebar, each agent has a role, department, tool stations, skills, memory, and approval policy. The office layout is functional: moving agents to stations changes what they can do. And it does not fake output. If the backend is offline, agent work is blocked until a real CrewAI or SimAI-compatible runtime connects."

## 2-minute demo

1. Open the app and show the command rail.
2. Select the CEO agent.
3. Open the right-side inspector.
4. Show role, model, skills, station grants, memory, and workflow tabs.
5. Select a furniture station and show its capability grants.
6. Start the backend and show the runtime switch to `CREWAI`.
7. Run an executive brief.
8. Switch to the Social Media Manager.
9. Run a campaign draft from the broadcast station.
10. Explain the no-fake-output design principle.

## YouTube / TikTok hook

"Most AI agent demos hide the agents inside a chat box. I built a 3D office where every agent has a role, workstation, tools, memory, and approval policy. And if the backend is offline, it refuses to fake the work."

## Product Hunt tagline

A 3D command center for managing AI agent teams, tools, memory, workflows, and workstations.

## GitHub launch post

SimOffice is open source: a 3D command center for AI agent teams. It uses React Three Fiber, FastAPI, and CrewAI to turn agents into visible operators with roles, departments, furniture-based tool grants, memory controls, workflows, and real runtime execution. No fake outputs. No invisible magic. Just a practical control plane for agentic operations.
